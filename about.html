ESE Project VI - About

This page includes a brief overview of Project VI for ESE!
For this project, we will be working on designing and implementing a
system capable of controlling and storing data for an elevator (which
in this case is completely remote). This includes a fully functioning
CAN network, along with the ability to control it via a web user
interface.
